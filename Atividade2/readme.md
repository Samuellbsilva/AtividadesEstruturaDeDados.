# Prática de Programação: Lógica e Algoritmos

Este repositório contém a resolução da segunda lista de exercícios de lógica. Os algoritmos foram desenvolvidos com foco em manipulação aritmética de tempo e verificação de integridade em coleções de dados.

## 🚀 Exercícios Implementados

1. **Conversor de Tempo:**
   - O programa recebe um valor inteiro em segundos e realiza a conversão para o formato de Horas, Minutos e Segundos.
   - Utiliza operações de divisão inteira (`//`) e resto da divisão (`%`) para extrair as unidades de tempo.

2. **Verificador de Elementos Únicos:**
   - Uma função que percorre um vetor de inteiros e verifica se existe algum elemento duplicado.
   - A lógica utiliza uma estrutura de repetição aninhada para contar a ocorrência de cada item, garantindo a validação da lista.
