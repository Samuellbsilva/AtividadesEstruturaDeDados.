# Atividade de Algoritmos - Python

Este repositório contém a resolução de três exercícios de lógica de programação propostos para a disciplina. Os algoritmos foram implementados em Python, focando em estruturas de repetição, condicionais e manipulação de listas e strings.

## 🚀 Exercícios Implementados

1. **Ranking de Notas:**
   - Cálculo de média aritmética de um vetor de notas.
   - Contagem de alunos acima da média.
   - Busca manual da maior nota (sem funções de ordenação).

2. **Simulador de Caixa Eletrônico:**
   - Cálculo da menor quantidade de cédulas (R$ 50, R$ 20 e R$ 10).
   - Uso de operadores de divisão inteira e resto.

3. **Validador de Senha:**
   - Verificação de comprimento mínimo (8 caracteres).
   - Verificação de presença de caracteres especiais.

## 🛠️ Tecnologias Utilizadas

- **Linguagem:** Python 3.x
- **Paradigma:** Estruturado/Imperativo
