# Atividade de Lógica de Programação - Java

Este repositório contém a implementação de cinco algoritmos fundamentais desenvolvidos em Java. O projeto foca na aplicação de estruturas de controle, manipulação de arrays e lógica aritmética básica.

## 🚀 Exercícios Implementados

1. **Ranking de Notas:**
   - Processamento de um vetor estático de notas para calcular a média.
   - Identificação da maior nota e contagem de notas acima da média sem o uso de bibliotecas de ordenação.

2. **Simulador de Caixa Eletrônico:**
   - Cálculo da distribuição mínima de cédulas (R$ 50, R$ 20 e R$ 10) utilizando operadores de divisão e resto.

3. **Validador de Senha Segura:**
   - Verificação de critérios de segurança: mínimo de 8 caracteres e presença de símbolos especiais.

4. **Conversor de Tempo:**
   - Conversão de um valor inteiro em segundos para o formato composto (Horas, Minutos e Segundos).

5. **Verificador de Elementos Duplicados:**
   - Algoritmo que identifica se há números repetidos em um array utilizando varredura aninhada.

## 🛠️ Tecnologias Utilizadas

- **Linguagem:** Java (JDK 8 ou superior)
- **Entrada de Dados:** `java.util.Scanner`
- **Ambiente:** Compiladores Online / IDEs Java
